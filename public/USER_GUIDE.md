## Earning Points

### How Points Work

#### Base Points System
Every event you attend earns you **10 base points**.

#### Bonus Points

**New Category Bonus** (+5 points)
- First time attending an event in a specific category
- Example: First academic event, first sports event, etc.

**First Event of the Month** (+15 points)
- Your first event attendance each calendar month
- Automatically applied

#### Category Multipliers

Different event categories have different point multipliers:

| Category | Multiplier | Example Points |
|----------|------------|----------------|
| Academic | 1.2x | 10 √ó 1.2 = 12 points |
| Social | 1.0x | 10 √ó 1.0 = 10 points |
| Sports | 1.0x | 10 √ó 1.0 = 10 points |
| Community Service | 1.5x | 10 √ó 1.5 = 15 points |
| Cultural | 1.1x | 10 √ó 1.1 = 11 points |

### Points Calculation Example

**Scenario**: You attend a Community Service event for the first time in November.
```
Base Points:           10
New Category Bonus:    +5
Subtotal:              15
Multiplier (1.5x):     15 √ó 1.5 = 22.5
Final Points:          23 points (rounded)
```

## Redeeming Rewards

### Redemption Process

#### Step 1: Select Reward
1. Find a reward you want
2. Ensure you have enough points
3. Click **"Redeem"** button

#### Step 2: Confirmation Modal
Review the redemption details:
- Reward image and description
- Points cost
- Your current balance
- Points remaining after redemption

**Warning Message**: If redemption will use most of your points (leaving <50), you'll see a warning message.

#### Step 3: Confirm
- Click **"Confirm Redemption"** to proceed
- Click **"Cancel"** to go back

#### Step 4: Receive Code
After successful redemption:
- A unique redemption code is generated
- Code format: `RC-TIMESTAMP-RANDOMCODE`
- Example: `RC-1700567890-ABC123XYZ`

### Using Your Redemption Code

1. **Copy the Code**: Click the "Copy" button
2. **Save Screenshot**: Recommended for your records
3. **Check Email**: You'll receive confirmation with redemption instructions
4. **Present Code**: Show code at redemption location (varies by reward)

### Redemption Locations

| Reward Type | Where to Redeem |
|-------------|-----------------|
| Campus Merchandise | Campus bookstore |
| Food Gift Cards | Participating restaurants |
| Gym Passes | Campus recreation center |
| Digital Rewards | Online (instructions in email) |
| Parking Passes | Parking services office |

### Important Notes

- ‚úÖ Redemption codes are **single-use only**
- ‚úÖ Codes **expire in 30 days**
- ‚úÖ Points are **deducted immediately** upon redemption
- ‚úÖ Redemptions are **non-refundable**
- ‚úÖ Save your code - you can't retrieve it later

## Tips & Best Practices

### Maximizing Points

#### Short Term (Weekly)
- Attend at least 2-3 events per week
- Focus on community service events (1.5x multiplier)
- Try a new event category each week

#### Medium Term (Monthly)
- Attend at least one event on the 1st of each month (+15 bonus)
- Participate in all 5 event categories
- Aim for 8-12 events per month

#### Long Term (Semester)
- Set a point goal (e.g., 1000 points by semester end)
- Track progress weekly
- Save points for high-value rewards

### Reward Strategy

#### Beginner Strategy (0-500 points)
- Redeem small rewards to get started
- Try cheap items (100-200 points)
- Learn the system

#### Intermediate Strategy (500-1500 points)
- Save for medium rewards (300-500 points)
- Keep at least 200 points in reserve
- Balance redemption with accumulation

#### Advanced Strategy (1500+ points)
- Target premium rewards (500-700 points)
- Maintain healthy point balance
- Strategic redemption timing

### Common Mistakes to Avoid

‚ùå **Spending all points immediately**
- Keep a reserve for better rewards
- Premium items offer better value

‚ùå **Ignoring event diversity**
- New category bonuses are significant
- Don't stick to just one type

‚ùå **Forgetting to check rewards before earning**
- Know what you're working toward
- Set specific reward goals

‚ùå **Missing first-of-month events**
- The +15 bonus is substantial
- Mark calendar for early month events

## FAQs

### Account & Access

**Q: How do I create an account?**
A: Use your existing TMU student credentials. No separate registration needed.

**Q: I forgot my password. What do I do?**
A: Use your TMU password recovery system.

**Q: Can I have multiple accounts?**
A: No, one account per student based on student ID.

### Points

**Q: When do I receive points after attending an event?**
A: Points are awarded automatically within 24 hours of event attendance verification.

**Q: Do points expire?**
A: No, earned points never expire. However, redemption codes expire in 30 days.

**Q: Can I transfer points to another student?**
A: No, points are non-transferable and tied to your student account.

**Q: What if I don't receive points after attending an event?**
A: Contact event organizers or system support with event details.

**Q: Can I lose points?**
A: Points are only deducted when redeeming rewards. There are no penalties.

### Rewards

**Q: How often is the rewards catalog updated?**
A: New rewards are added monthly. Out-of-stock items are restocked regularly.

**Q: What if my desired reward is out of stock?**
A: Check back regularly. You can also contact support to request notifications when it's restocked.

**Q: Can I cancel a redemption?**
A: No, all redemptions are final. Choose carefully before confirming.

**Q: My redemption code isn't working. What should I do?**
A: Contact support immediately with your redemption code and reward details.

**Q: Can I redeem multiple rewards at once?**
A: Yes, as long as you have sufficient points for each redemption.

### Leaderboard

**Q: How often does the leaderboard update?**
A: The leaderboard updates in real-time as points are awarded.

**Q: Can I hide my name from the leaderboard?**
A: Contact support to request privacy settings (feature coming soon).

**Q: What happens to leaderboard rankings at semester end?**
A: All-time rankings persist. Monthly and weekly rankings reset on schedule.

## Troubleshooting

### Common Issues

#### Issue: Can't see my points after attending an event

**Solutions**:
1. Wait 24 hours for processing
2. Check Points History on dashboard
3. Verify event organizer confirmed attendance
4. Contact support if issue persists after 24 hours

#### Issue: Reward redemption button is disabled

**Possible Reasons**:
- Insufficient points (check balance)
- Reward out of stock (check badge)
- Reward inactive (temporarily unavailable)

**Solution**: Earn more points or choose a different reward

#### Issue: Redemption code not working

**Steps**:
1. Verify you copied the complete code (no spaces)
2. Check code hasn't expired (30-day limit)
3. Confirm you're at the correct redemption location
4. Contact support with code and reward details

#### Issue: Dashboard not loading

**Steps**:
1. Refresh page (F5 or Ctrl+R)
2. Clear browser cache
3. Try different browser
4. Check internet connection
5. Contact support if issue continues

#### Issue: Images not displaying

**Steps**:
1. Check internet connection
2. Disable browser extensions (ad blockers)
3. Try incognito/private browsing mode
4. Clear browser cache

#### Issue: Leaderboard shows incorrect rank

**Solutions**:
1. Wait a few minutes for update
2. Refresh page
3. Points may be pending (24-hour processing)
4. Contact support if rank doesn't update after 24 hours
